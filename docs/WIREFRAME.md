# 와이어프레임 (Wireframe)

와이어프레임은 실제 디자인 전, 모바일 화면의 정보 배치와 이동 흐름을 빠르게 확인하기 위한 문서입니다.

## 1. 공통 레이아웃

대상 디바이스:
- 부모님 갤럭시 안드로이드
- 제작자 iPhone 테스트

기본 구조:
- 상단: 화면 제목 또는 여행 상태
- 본문: 현재 화면의 핵심 정보
- 하단: 4개 탭 메뉴

하단 탭:
- 오늘
- 전체 일정
- 지도
- 긴급

```text
┌─────────────────────┐
│ 화면 제목 / 상태       │
├─────────────────────┤
│                     │
│ 핵심 콘텐츠           │
│                     │
├─────────────────────┤
│ 오늘 전체일정 지도 긴급 │
└─────────────────────┘
```

## 2. 오늘 화면 (홈)

목적:
- 부모님이 앱을 열자마자 오늘 할 일과 다음 행동을 확인한다.

구성:
- 여행 제목과 기간
- 다음 일정 카드 (시간 + 행동 + 지도 버튼)
- 일정 보기 / 긴급 연락 바로가기
- 추천 루트 요약

```text
┌─────────────────────┐
│ 가고시마 여행          │
│ 6월 27일(토) ~ 30일(화) │
├─────────────────────┤
│ 다음 일정              │
│ 오전 골프 라운딩 1일차   │
│ 전날 밤 장갑/골프화 챙기기│
│ [지도 열기]            │
├─────────────────────┤
│ [일정 보기] [긴급 연락]  │
├─────────────────────┤
│ 추천 루트              │
│ 숙소 ↔ 골프장 왕복 루트  │
├─────────────────────┤
│ 오늘 전체일정 지도 긴급  │
└─────────────────────┘
```

## 3. 전체 일정 화면

목적:
- 3박 4일 전체 일정을 날짜별로 확인하고, 출발 전 준비물을 함께 점검한다.

구성:
- 날짜 선택 탭
- 일정 카드 목록 (시간, 유형, 장소, 이동/예약 메모)
- 카테고리별 체크리스트

```text
┌─────────────────────┐
│ 일정                 │
├─────────────────────┤
│ [6/27] [6/28] [6/29] │
├─────────────────────┤
│ 오전 골프              │
│ 골프 라운딩 1일차       │
│ 예약: 티오프 시간 확인   │
│ [지도]                │
├─────────────────────┤
│ 저녁 식사              │
│ 라운딩 후 저녁 식사      │
│ 메모: 무리하지 말고 이동 │
├─────────────────────┤
│ 준비 체크리스트          │
│ 출발 전 ☑ 여권 ☐ 골프화  │
├─────────────────────┤
│ 오늘 전체일정 지도 긴급  │
└─────────────────────┘
```

## 4. 지도 화면

목적:
- 숙소, 골프장, 식당, 관광지 등 핵심 장소를 한눈에 보고 Google Maps로 바로 이동한다.

구성:
- 장소 핀 미리보기
- 카테고리별 장소 카드 (추천 이유, 주의사항, 지도 버튼)

```text
┌─────────────────────┐
│ 지도와 추천 장소        │
├─────────────────────┤
│   ⌖        ⌖        │
│        ⌖             │
├─────────────────────┤
│ [골프] 골프장          │
│ 라운딩 장소입니다        │
│ 주의: 장갑/골프화 준비   │
│ [보기]                │
├─────────────────────┤
│ [식사] 흑돼지 맛집 후보  │
│ 가고시마 대표 음식       │
│ [보기]                │
├─────────────────────┤
│ 오늘 전체일정 지도 긴급  │
└─────────────────────┘
```

## 5. 긴급 화면

목적:
- 문제가 생겼을 때 필요한 연락처와 여행 정보를 한 화면에서 바로 확인한다.

구성:
- 가족/숙소 연락 (전화 버튼)
- 여권 분실 시 행동 순서
- 항공편, 숙소, 일본어 문장 등 여행 정보

```text
┌─────────────────────┐
│ 긴급과 여행 정보        │
├─────────────────────┤
│ 가족에게 전화          │
│ [전화하기]             │
├─────────────────────┤
│ 숙소                  │
│ 호텔명 / 전화번호       │
│ [전화하기]             │
├─────────────────────┤
│ 여권을 잃어버렸다면     │
│ 1. 가족에게 연락       │
│ 2. 가까운 경찰서 방문   │
│ 3. 영사관 연락         │
├─────────────────────┤
│ 여행 정보              │
│ 항공편 / 숙소 / 일본어  │
├─────────────────────┤
│ 오늘 전체일정 지도 긴급  │
└─────────────────────┘
```

## 6. 와이어프레임 검증 기준

- 부모님이 첫 화면에서 다음 행동을 바로 찾을 수 있는가?
- 하단 탭 이름이 짧고 이해하기 쉬운가? (4개로 압축)
- 긴급 연락과 여행 정보가 한 화면에서 바로 접근 가능한가?
- 지도 열기, 전화, 복사 버튼이 충분히 큰가?
- 화면 하나에 정보가 과도하게 많지 않은가?
- iPhone과 갤럭시 화면에서 텍스트가 잘리지 않는가?
