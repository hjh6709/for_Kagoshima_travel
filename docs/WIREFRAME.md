# 와이어프레임 (Wireframe)

와이어프레임은 실제 디자인 전, 모바일 화면의 정보 배치와 이동 흐름을 빠르게 확인하기 위한 문서입니다.

## 1. 공통 레이아웃

대상 디바이스:
- 부모님 갤럭시 안드로이드
- 제작자 iPhone 테스트

기본 구조:
- 상단: 화면 제목 또는 여행 상태
- 본문: 현재 화면의 핵심 정보
- 하단: 5개 탭 메뉴

하단 탭:
- 홈
- 일정
- 정보
- 체크
- 긴급

```text
┌─────────────────────┐
│ 화면 제목 / 상태       │
├─────────────────────┤
│                     │
│ 핵심 콘텐츠           │
│                     │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급 │
└─────────────────────┘
```

## 2. 홈 화면

목적:
- 부모님이 앱을 열자마자 오늘 할 일과 다음 행동을 확인한다.

구성:
- 여행 제목
- 오늘 날짜와 여행 일차
- 다음 일정 카드
- 오늘 일정 요약
- 긴급 버튼

```text
┌─────────────────────┐
│ 가고시마 여행          │
│ 6월 00일 / 2일차       │
├─────────────────────┤
│ 다음 일정              │
│ 10:30 센간엔 이동      │
│ 전차 + 도보            │
│ [지도 열기] [상세 보기] │
├─────────────────────┤
│ 오늘 일정              │
│ 09:00 조식             │
│ 10:30 센간엔           │
│ 13:00 점심             │
├─────────────────────┤
│ [긴급 연락 바로가기]    │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급  │
└─────────────────────┘
```

## 3. 일정 화면

목적:
- 날짜별 일정을 시간순으로 확인한다.

구성:
- 날짜 선택
- 일정 카드 목록
- 유형 아이콘
- 지도 버튼

```text
┌─────────────────────┐
│ 일정                 │
├─────────────────────┤
│ [6/27] [6/28] [6/29] │
├─────────────────────┤
│ 09:00 식사            │
│ 호텔 조식             │
│ 메모: 천천히 출발      │
├─────────────────────┤
│ 10:30 관광            │
│ 센간엔                │
│ [지도] [상세]          │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급  │
└─────────────────────┘
```

## 4. 일정 상세 화면

목적:
- 특정 일정에 필요한 주소, 이동, 예약 메모를 확인한다.

구성:
- 일정 제목
- 시간과 유형
- 장소명과 주소
- 이동 방법
- 부모님께 보여줄 메모
- 지도, 주소 복사 버튼

```text
┌─────────────────────┐
│ < 일정 상세           │
├─────────────────────┤
│ 센간엔 관광            │
│ 10:30 / 관광          │
├─────────────────────┤
│ 장소                  │
│ 센간엔                │
│ 주소                  │
│ 일본어 주소 표시       │
│ [지도 열기] [주소 복사] │
├─────────────────────┤
│ 이동 방법              │
│ 전차 하차 후 도보       │
├─────────────────────┤
│ 부모님 메모            │
│ 입장권 확인 후 입장     │
└─────────────────────┘
```

## 5. 여행 정보 화면

목적:
- 항공편, 숙소, 예약 정보를 한곳에서 확인한다.

구성:
- 항공편 섹션
- 숙소 섹션
- 예약번호
- 일본어 문장

```text
┌─────────────────────┐
│ 여행 정보             │
├─────────────────────┤
│ 항공편                │
│ 출국: 항공편 / 시간    │
│ 귀국: 항공편 / 시간    │
├─────────────────────┤
│ 숙소                  │
│ 호텔명                │
│ 주소                  │
│ [지도] [전화] [복사]   │
├─────────────────────┤
│ 일본어 문장            │
│ 이 주소로 가주세요      │
│ [복사]                │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급  │
└─────────────────────┘
```

## 6. 체크리스트 화면

목적:
- 준비물과 여행 중 확인할 일을 체크한다.

구성:
- 카테고리 필터
- 체크 항목
- 완료 개수 표시

```text
┌─────────────────────┐
│ 체크리스트            │
│ 8개 중 5개 완료        │
├─────────────────────┤
│ [출국 전] [공항] [매일]│
├─────────────────────┤
│ ☑ 여권                │
│ ☑ 항공권              │
│ ☐ 보조배터리           │
│ ☐ 여행자보험 확인      │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급  │
└─────────────────────┘
```

## 7. 긴급 화면

목적:
- 문제가 생겼을 때 필요한 연락처와 행동 순서를 바로 확인한다.

구성:
- 가족 연락
- 숙소 연락
- 보험/영사관
- 상황별 행동 순서

```text
┌─────────────────────┐
│ 긴급                 │
├─────────────────────┤
│ 가족에게 전화          │
│ [전화하기]             │
├─────────────────────┤
│ 숙소                  │
│ 호텔명 / 전화번호       │
│ [전화] [주소 복사]      │
├─────────────────────┤
│ 여권을 잃어버렸다면     │
│ 1. 가족에게 연락       │
│ 2. 가까운 경찰서 방문   │
│ 3. 영사관 연락         │
├─────────────────────┤
│ 홈 일정 정보 체크 긴급  │
└─────────────────────┘
```

## 8. 와이어프레임 검증 기준

- 부모님이 첫 화면에서 다음 행동을 바로 찾을 수 있는가?
- 하단 탭 이름이 짧고 이해하기 쉬운가?
- 긴급 연락 화면이 항상 접근 가능한가?
- 지도 열기, 전화, 복사 버튼이 충분히 큰가?
- 화면 하나에 정보가 과도하게 많지 않은가?
- iPhone과 갤럭시 화면에서 텍스트가 잘리지 않는가?
